📦 Patching & Downloads

Download the patched builds below and place them in the correct locations to connect to this backend. Each card shows exactly where the file belongs on your system.

🎮 Client Patch — Play on this backend

Replace your existing BeamMP launcher with the patched version below. It is pre-configured to connect to this backend instead of the official servers. The mod is automatically downloaded by the launcher.

💡 Alternative: No Patch Needed

You can also use the BeamNG Content Manager instead of patching the launcher. Simply change the auth URL and backend URL in the Content Manager's settings to point to this backend, and you'll be able to browse and connect to servers here without any additional client patch.

🖥️ Server Setup — Host a game server

Download the server binary for your platform. Place it in a dedicated folder, configure ServerConfig.toml with your auth key from the My Server Keys section, then run the executable.
